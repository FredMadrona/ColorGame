How to Play
The user has an initial balance of 100. This balance can be used to bet on the six (6) colors. The user wins his/her bet everytime the color appears on the winning combinations above. once - x2 double - x3 triple - x4 . If not, the bet will be subtracted to the users balance. The game will continue as long as the player has remaining balance.
